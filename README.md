## Run the project
### Get the code
```shell
$ git clone https://github.com/Ilyes-Hammadi/hill-image-cipher.git
$ cd hill-image-cipher
```

### Install dependencies
```shell
$ pip install -r requirements.txt
```

### Encode an image
```shell
$ python main.py image.jpg
```
